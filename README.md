# Counter (React + Redux + TypeScript)

<h3> To run this project, follow the commands: </h3>
<br />
  
```
npm install
```

And then:

```
npm run start
```

<br />

## Concepts used in this project

<br />

-   React Components
-   Redux (w/ TypeScript setup)
-   Material UI
-   TypeScript
-   CSS-in-JS
